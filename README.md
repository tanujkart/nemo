# Abyssal Mission Control

High-fidelity underwater mission control concept designed to feel cinematic, immersive, and memorable while staying readable.

## Structure

- `index.html` - main app shell
- `src/styles.css` - visual system and layout
- `src/data.js` - UI data
- `src/app.js` - rendering and lightweight behavior

## Run locally

From this directory:

```bash
npm run dev
```

Then open:

- [http://localhost:4173](http://localhost:4173)

## Design intent

- Central hybrid command core for strong visual focus
- Atmospheric underwater depth with holographic layers
- Less corporate dashboard framing, more mission-HUD immersion
- Practical information hierarchy for quick comprehension
